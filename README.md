# Jaemin Oh's Website


## Commands

export conda env: `conda env export | grep -v "^prefix: " > environment.yml`

view the site locally: `mkdocs serve`

publish the site: `mkdocs gh-deploy --force`

script for publish the site: `sh publish.sh`
